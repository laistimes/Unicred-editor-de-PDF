Padrão ISO para arquivamento digital. Garante que o documento esteja em conformidade com a ISO 19005. Esta ferramenta injeta os metadados XMP de conformidade PDF/A-1b no seu arquivo.

📝 PDF para Word

Extrai o texto do PDF e gera um arquivo .docx editável no Word, LibreOffice ou Google Docs.

⚠️
Imagens não são convertidas
Somente o texto do PDF é extraído. Imagens, gráficos e elementos visuais não aparecem no documento Word gerado.
Se o PDF for composto só de imagens (ex: scan), o resultado pode ficar vazio.

📊 PDF para PowerPoint

Converte cada página do PDF em um slide. O visual é 100% preservado — cada página vira uma imagem no slide.

ℹ️
Como funciona
Cada página é renderizada como imagem e inserida em um slide. O layout visual é preservado exatamente como no original. O texto nos slides não será editável — para editar texto use PDF → Word.
